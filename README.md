# quicklisp-quackload

This project adds duck typing to the Quicklisp client.

## Exports

* `QL:QUACKLOAD` - behaves just like `QL:QUICKLOAD`, except with duck typing.

## Example

```common-lisp
CL-USER> (ql:quackload :drakma)
Quack quack "quack":
  Quack 1 QUACK quack:
    quack

; Quack "quack"
[quack quack]....................................
[quack quack-quack64]...............................
[quack quack-quack]................................
..................................................
[quack quack]...................................
[quack quack].................................
[quack quack+quack]..................................
[quack quack].......
(:DRAKMA)
```
